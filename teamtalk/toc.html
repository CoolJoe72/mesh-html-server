TeamTalk 4 Help

This help file contains information on how to use the TeamTalk application and setting up a TeamTalk server.

TeamTalk 4 Classic for Accessibility

Users using TeamTalk 4 Classic for Accessibility should note that this help file is intended for the Qt-based TeamTalk 4 user interface so some options may appear different in the Classic version. For more information on the accessible version of TeamTalk 4 please read the section on accessibility.
